Termos de uso

1. Introdução

Bem-vindo ao nosso chat!

2. Uso do chat

O nosso chat é um espaço para interação e conversa entre os participantes. Não é necessário realizar nenhum tipo de cadastro para utilizar o chat.

3. Comportamento dos usuários

Para manter um ambiente saudável e respeitoso, pedimos que todos os usuários sigam as seguintes diretrizes:

  • Seja cortês e respeitoso com os outros participantes;
  • Evite o uso de linguagem ofensiva, discriminatória ou que viole os direitos dos outros;
  • Não compartilhe informações pessoais sensíveis, como números de telefone, endereços, senhas, etc.;
  • Esteja ciente de que todas as mensagens enviadas no chat são visíveis para todos os participantes;
  • Não compartilhe conteúdo protegido por direitos autorais sem permissão;
  • Evite o envio excessivo de mensagens ou spam.

4. Responsabilidade

O uso do nosso chat é de responsabilidade do usuário. Não nos responsabilizamos por qualquer conteúdo compartilhado pelos participantes.
